# Projet-Meteo-and-Trend
__OBJECTIF__ : Analyser des données brutes et les rendres visuellement compréhensibles

__Cahier des charges__ : 
1. Récupération : Se connecter à l'API OpenWeatherMap pour récupérer les prévisions sur 7 jours d'une ville donnée.
2. Traitement : Nettoyer les données (ex: convertir les Kelvin en Celsius, formater les dates).
3. Visualisation : Générer un graphique de la courbe des températures et un diagramme en barres pour les précipitations.
4. Export : Sauvegarder automatiquement le graphique en format .png avec la date du jour.

__Technologies__ :
Langage : Python
Requêtes : Bibliothèque requests pour l'API.
Analyse : Pandas (la référence en Data Science).
Graphiques : Matplotlib ou Seaborn.
